# Taller de : adapta el tamaño de la fuente según el ancho de la pantalla

RECORDATORIO TEÓRICO El diseño responsivo no se limita a reorganizar bloques en la pantalla: también implica adaptar la tipografía para que sea legible en cualquier dispositivo. Existen tres enfoques principales para lograr fuentes responsivas con CSS3. Primero, las unidades relativas: 'em' es relativa al tamaño de fuente del elemento padre, mientras que 'rem' es relativa al tamaño de fuente del elemento raíz (html). Usar rem facilita la coherencia global. Segundo, las media queries permiten redefinir el font-size en función de breakpoints específicos: por ejemplo, establecer 14px en móvil, 16px en tableta y 18px en escritorio. Tercero, la función clamp() de CSS moderno permite una interpolación fluida: clamp(valor-mínimo, valor-preferido, valor-máximo), donde el valor preferido suele expresarse en unidades viewport como vw. Ejemplo: font-size: clamp(14px, 2vw, 20px) significa que el tamaño oscilará entre 14px y 20px según el ancho de la ventana, sin saltos bruscos. Error frecuente: definir todos los tamaños en px fijos hace que el texto sea ilegible en pantallas pequeñas o demasiado grande en pantallas anchas. Otro error habitual es olvidar aplicar las reglas al elemento html o body antes de usar rem en los elementos hijos. Modelo mental: piensa en la tipografía responsiva como un sistema de escalas, no como valores aislados.

# Taller: maquetación del blog (barra lateral + contenido)

**Recordatorio teórico** CSS Grid es un sistema de maquetación bidimensional que permite organizar elementos en filas y columnas simultáneamente. A diferencia de Flexbox, que trabaja en un único eje, Grid es ideal para estructuras complejas como la clásica disposición de un blog con barra lateral. Los conceptos clave que necesitarás en este taller son: el contenedor de cuadrícula (display: grid), las pistas de columna y fila (grid-template-columns, grid-template-rows), las áreas nombradas (grid-template-areas) y las unidades fraccionarias (fr). Recuerda que grid-template-areas permite asignar nombres semánticos a las zonas del layout, lo que facilita enormemente la lectura y el mantenimiento del código. Un error frecuente en principiantes es olvidar que cada hijo directo del contenedor grid se convierte automáticamente en un grid item; no es necesario añadir propiedades adicionales salvo que quieras controlar su posición explícitamente. Otro error habitual es mezclar unidades fijas (px) con unidades relativas (fr, %) sin coherencia, lo que rompe el comportamiento responsivo. Mini-ejemplo: para crear dos columnas donde la barra lateral ocupa 1 fracción y el contenido principal 3 fracciones, basta con escribir grid-template-columns: 1fr 3fr en el contenedor. Combinado con media queries, esta misma cuadrícula puede apilarse en una sola columna en móviles. --- **Presentación del taller** En este taller pondrás en práctica los conocimientos adquiridos sobre CSS Grid, diseño responsivo y unidades relativas para construir la maquetación completa de una página de blog. El objetivo concreto es desarrollar una interfaz compuesta por una cabecera, un área de contenido principal, una barra lateral, y un pie...

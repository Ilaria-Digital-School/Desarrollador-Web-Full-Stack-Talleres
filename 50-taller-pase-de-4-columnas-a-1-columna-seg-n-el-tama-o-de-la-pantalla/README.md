# Taller: pase de 4 columnas a 1 columna según el tamaño de la pantalla

RECORDATORIO TEÓRICO El diseño responsivo con CSS Grid se apoya en la capacidad de redefinir la estructura de la cuadrícula según el tamaño de la pantalla mediante media queries. Una cuadrícula de 4 columnas en escritorio puede y debe transformarse en una sola columna en móvil para garantizar la legibilidad y la usabilidad. El modelo mental clave es el siguiente: defines primero una cuadrícula base (generalmente para escritorio o para móvil, según el enfoque mobile-first o desktop-first), y luego sobrescribes las propiedades de la cuadrícula dentro de bloques @media. Las propiedades más utilizadas en este contexto son grid-template-columns, gap y, eventualmente, grid-column para reposicionar elementos individuales. Un error frecuente es olvidar que, al pasar a 1 columna, los elementos hijos que tenían asignadas posiciones explícitas (grid-column: span 2, por ejemplo) deben ser reajustados para evitar desbordamientos o espacios vacíos. Otro error habitual es definir los puntos de ruptura (breakpoints) de forma arbitraria en lugar de basarse en el contenido real. Mini-ejemplo: .grid { display: grid; grid-template-columns: repeat(4, 1fr); gap: 1rem; } @media (max-width: 768px) { .grid { grid-template-columns: 1fr; } } Este fragmento convierte una cuadrícula de 4 columnas en una de 1 columna cuando el ancho de la pantalla es igual o inferior a 768px.

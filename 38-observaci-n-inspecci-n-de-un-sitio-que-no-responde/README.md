# Observación : inspección de un sitio que no responde

RECORDATORIO TEÓRICO El diseño responsivo es una filosofía de desarrollo web que permite que una interfaz se adapte fluidamente a cualquier tamaño de pantalla, desde un teléfono móvil hasta un monitor de escritorio de gran resolución. Tres conceptos son fundamentales: la ventana gráfica (viewport), que representa el área visible del navegador en el dispositivo del usuario; los puntos de interrupción (breakpoints), que son umbrales de ancho a partir de los cuales el diseño se reorganiza mediante media queries CSS; y la fluidez, que describe la capacidad de los elementos de cambiar de tamaño de forma proporcional en lugar de permanecer fijos en píxeles. Un error frecuente entre los principiantes es confundir un diseño 'estrecho' con un diseño responsivo: reducir el ancho del contenedor no es suficiente si las imágenes, los textos o los contenedores internos siguen teniendo dimensiones fijas. Otro error clásico es olvidar la etiqueta meta viewport en el HTML (), lo que hace que el navegador móvil simule una pantalla de escritorio y comprima todo el contenido. Mini-ejemplo mental: imagina una tabla HTML con anchos de columna fijados en píxeles (width: 400px).

# Taller: aplicar correctamente los niveles de título

**Recordatorio teórico** La jerarquía de títulos en HTML va del H1 al H6 y representa la estructura lógica y semántica del contenido de una página. El H1 es el título principal de la página y debe aparecer una sola vez por documento; los niveles siguientes (H2, H3, H4...) organizan los subtemas de forma descendente, como los capítulos y subcapítulos de un libro. Esta jerarquía no es solo visual: los motores de búsqueda la utilizan para indexar el contenido y las tecnologías de asistencia (lectores de pantalla) la usan para que los usuarios con discapacidad visual puedan navegar de forma eficiente por la página. Un error muy frecuente entre los principiantes es elegir el nivel de título según su tamaño visual en lugar de su peso semántico. Por ejemplo, usar un H3 directamente después de un H1 por razones estéticas rompe la jerarquía lógica. Otro error habitual es utilizar múltiples H1 en una misma página o no utilizar títulos en las secciones semánticas importantes (como `` o ``).

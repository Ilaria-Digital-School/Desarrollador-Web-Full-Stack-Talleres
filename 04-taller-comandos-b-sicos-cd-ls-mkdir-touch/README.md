# Taller: comandos básicos (cd, ls, mkdir, touch)

Recordatorio teórico (≈200 palabras). En cualquier sistema operativo basado en Unix, incluido macOS y la mayoría de distribuciones Linux, la terminal es la puerta de entrada a la comunicación directa con el sistema. cd (change directory) cambia el directorio de trabajo, ls (list) enumera el contenido del directorio actual, mkdir (make directory) crea carpetas y touch crea archivos vacíos o actualiza la marca de tiempo de los existentes. Un error habitual consiste en perder la noción de la ruta actual y, por ello, crear archivos o carpetas en lugares incorrectos; otro es olvidar las comillas cuando los nombres contienen espacios. Mantén un modelo mental claro: imagina el árbol de carpetas como un mapa en el que cd mueve el “punto rojo” que indica tu posición, ls te muestra lo que hay en ese punto, mkdir añade una calle nueva y touch coloca un cartel. Mini-ejemplo: desde tu carpeta personal (~), ejecuta mkdir proyecto && cd proyecto && touch README.md; con tres órdenes breves habrás creado una carpeta, te habrás desplazado a ella y habrás generado un archivo.

# Taller: comandos básicos (cd, ls, mkdir, touch)

**Recordatorio teórico** La terminal es una interfaz de texto que permite comunicarse directamente con el sistema operativo mediante comandos escritos. A diferencia de un explorador de archivos gráfico, la terminal no tiene botones ni íconos: todo se hace escribiendo instrucciones precisas. Este modo de trabajo es fundamental en el desarrollo web profesional porque permite automatizar tareas, gestionar proyectos, interactuar con servidores remotos y usar herramientas como Git, Node.js o gestores de paquetes. Cuatro comandos esenciales que debes dominar desde ahora: - **ls** (list): muestra el contenido del directorio actual. - **cd** (change directory): permite navegar entre carpetas. `cd ..` sube un nivel; `cd nombre_carpeta` entra en una carpeta. - **mkdir** (make directory): crea una nueva carpeta. Ejemplo: `mkdir mi_proyecto`. - **touch**: crea un archivo vacío. Ejemplo: `touch index.html`. Error frecuente: confundir la ubicación actual en el sistema de archivos. Antes de ejecutar cualquier comando, usa `pwd` (print working directory) para saber exactamente dónde estás. Otro error habitual es olvidar las mayúsculas/minúsculas en los nombres de archivos y carpetas: en sistemas Unix/Linux/macOS, `Proyecto` y `proyecto` son entidades distintas. Mini-ejemplo: si estás en `/home/usuario` y ejecutas `mkdir taller`, luego `cd taller`, habrás creado una carpeta y entrado en ella. Un `touch archivo.txt` dentro crea un archivo vacío listo para usar. --- **Enunciado del taller** En este taller pondrás en práctica los comandos básicos de la terminal que has estudiado en la lección teórica anterior. El objetivo es que seas capaz de navegar por el sistema de archivos, crear una estructura de carpetas...

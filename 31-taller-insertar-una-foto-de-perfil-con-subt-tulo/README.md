# Taller: insertar una foto de perfil con subtítulo

RECORDATORIO TEÓRICO En HTML, la etiqueta permite incrustar una imagen en una página web. Es un elemento vacío (sin etiqueta de cierre) que requiere dos atributos esenciales: src, que indica la ruta hacia el archivo de imagen (relativa o absoluta), y alt, que proporciona un texto alternativo descriptivo para la accesibilidad y el SEO. Omitir el atributo alt es uno de los errores más frecuentes entre los principiantes, y tiene consecuencias reales para los usuarios con lectores de pantalla. Para enriquecer semánticamente una imagen y asociarle un subtítulo, HTML5 introduce las etiquetas y . La etiqueta actúa como contenedor semántico que agrupa la imagen con su descripción. La etiqueta contiene el texto del subtítulo y debe colocarse dentro del bloque , ya sea antes o después del . Este conjunto comunica claramente al navegador y a los motores de búsqueda que la imagen y su leyenda están relacionadas. Recuerda que src="./images/foto.jpg" es una ruta relativa, mientras que src="https://ejemplo.com/foto.jpg" es una ruta absoluta. --- ENUNCIADO DEL TALLER Este taller tiene como objetivo consolidar tu dominio de la etiqueta y de las estructuras semánticas y vistas en el capítulo anterior.

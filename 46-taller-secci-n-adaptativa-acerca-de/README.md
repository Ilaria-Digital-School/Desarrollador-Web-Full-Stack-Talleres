# Taller: sección adaptativa «Acerca de»

RECORDATORIO TEÓRICO El diseño responsivo consiste en crear interfaces que se adapten fluidamente a cualquier tamaño de pantalla sin duplicar código. CSS3 nos ofrece tres pilares fundamentales: las Media Queries, Flexbox y CSS Grid. Las Media Queries permiten aplicar estilos condicionalmente según el ancho del viewport (por ejemplo, @media (max-width: 768px) { ... }). Flexbox es ideal para distribuciones unidimensionales (fila o columna) y gestiona con precisión la alineación, el espaciado y el orden de los elementos. CSS Grid, por su parte, permite construir layouts bidimensionales donde se controlan filas y columnas simultáneamente. Modelo mental clave: piensa siempre en «mobile first». Define primero los estilos para pantallas pequeñas y luego amplía progresivamente con breakpoints. Esto evita sobreescrituras innecesarias y produce CSS más limpio. Errores frecuentes: olvidar la etiqueta meta viewport en el HTML (), mezclar Flexbox y Grid sin una estrategia clara, o usar píxeles fijos donde deberían usarse unidades relativas (%, em, rem, vw, vh). Mini-ejemplo: una sección con imagen y texto en fila en desktop que apila verticalmente en móvil: .about { display: flex; flex-direction: row; gap: 2rem; } @media (max-width: 768px) { .about { flex-direction: column; } } Este recordatorio es el puente directo hacia el taller que sigue.

# Taller: insertar encabezados y párrafos

RECORDATORIO TEÓRICO En HTML, la estructura de contenido textual se basa en dos tipos de elementos fundamentales: los encabezados y los párrafos. Los encabezados van del nivel h1 al h6 y establecen una jerarquía semántica en el documento. El h1 representa el título principal de la página, el h2 corresponde a secciones principales, el h3 a subsecciones, y así sucesivamente. Cada página debe tener un único h1, que resume su contenido global. Los párrafos se definen con la etiqueta p y contienen bloques de texto continuo. Un error frecuente entre los principiantes es usar los encabezados únicamente para cambiar el tamaño del texto (aplicando h1 porque 'se ve grande'), ignorando su valor semántico. Esto perjudica la accesibilidad y el posicionamiento en buscadores. Otro error habitual es no cerrar las etiquetas correctamente o anidar elementos de bloque dentro de un párrafo. Modelo mental clave: piensa en tu página como el índice de un libro. El h1 es el título del libro, los h2 son los capítulos, los h3 son las secciones dentro de cada capítulo. Los párrafos son el contenido narrativo entre esos títulos. Mini-ejemplo: Soy un desarrollador web en formación, aprendiendo HTML desde cero.

# Taller: creación de una galería de imágenes semánticas

RECORDATORIO TEÓRICO En HTML semántico, cada elemento debe tener un significado preciso que va más allá de su simple apariencia visual. Cuando trabajamos con imágenes, la etiqueta es el punto de partida, pero rara vez debe aparecer sola. El atributo alt es obligatorio: describe el contenido de la imagen para los lectores de pantalla y para los motores de búsqueda. Un alt vacío (alt="") indica que la imagen es puramente decorativa; un alt ausente es un error de accesibilidad grave. La etiqueta agrupa una imagen con su contenido relacionado (generalmente un pie de foto), mientras que proporciona ese pie de foto visible y semánticamente vinculado a la imagen. Este dúo figure/figcaption es la forma correcta y profesional de presentar imágenes con descripción en HTML5. Un error frecuente es usar un párrafo o un como sustituto de , lo que rompe la semántica del documento. Ejemplo mínimo correcto: Barcelona, España – Foto tomada desde el Tibidabo Otro error habitual es olvidar estructurar el conjunto de figuras dentro de una sección semántica (, , ), dejando los elementos flotando sin contexto en el documento.

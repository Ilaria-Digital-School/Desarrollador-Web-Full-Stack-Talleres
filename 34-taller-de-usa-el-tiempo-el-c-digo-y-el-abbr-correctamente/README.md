# Taller de : usa el tiempo, el código y el abbr correctamente

**Recordatorio teórico** Las etiquetas semánticas en línea ``, `` y `` tienen propósitos muy específicos y complementarios. La etiqueta `` sirve para marcar fechas, horas o duraciones de forma legible tanto para humanos como para máquinas; su atributo `datetime` contiene el valor normalizado (por ejemplo, `15 de marzo de 2024`). La etiqueta `` indica un fragmento de código informático dentro de un texto corriente, como un nombre de función, una variable o una instrucción: `console.log()`. La etiqueta `` señala una abreviatura o un acrónimo y usa el atributo `title` para proporcionar su forma completa: `HTML`. Los errores más frecuentes son: usar `` o `` en lugar de `` para destacar código; omitir el atributo `datetime` en `` (perdiendo así el valor semántico real); y olvidar el atributo `title` en ``, lo que anula completamente su utilidad. El modelo mental clave es este: si el navegador o un motor de búsqueda necesita interpretar el contenido de forma automática, la etiqueta debe llevar su atributo de precisión obligatorio.

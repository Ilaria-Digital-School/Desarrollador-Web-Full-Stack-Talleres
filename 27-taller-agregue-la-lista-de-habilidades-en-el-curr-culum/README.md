Existen tres tipos principales: la lista no ordenada (``), que agrupa elementos sin un orden jerárquico particular —ideal para listas de habilidades—; la lista ordenada (``), que implica una secuencia lógica o prioridad; y la lista de definición (``), útil para pares término-descripción. Cada elemento de una lista `` u `` se envuelve en una etiqueta ``. Usar listas en HTML no es solo una cuestión de presentación visual: es ante todo una decisión semántica. Error frecuente: usar saltos de línea (``) o párrafos (``) para simular una lista. Esto rompe la semántica del documento y perjudica la accesibilidad. Otro error habitual es anidar listas sin respetar la jerarquía correcta (un `` hijo debe estar dentro de un ``, no directamente dentro del `` padre).

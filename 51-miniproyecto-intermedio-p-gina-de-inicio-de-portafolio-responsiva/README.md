# Miniproyecto intermedio: página de inicio de portafolio responsiva

## Descripción del proyecto En este miniproyecto, crearás una página de inicio de portafolio profesional completamente responsiva utilizando HTML5 y CSS3, integrando de forma combinada Flexbox y CSS Grid. El objetivo es demostrar tu dominio de las técnicas de diseño responsivo aprendidas a lo largo del capítulo, produciendo una página funcional, visualmente coherente y adaptada a distintos tamaños de pantalla (móvil, tableta y escritorio). ## Contexto profesional Como desarrollador web en proceso de reconversión, necesitas ser capaz de construir interfaces responsivas desde cero. Una página de portafolio es además una herramienta profesional directamente reutilizable en tu búsqueda de empleo. Este proyecto te permite validar competencias técnicas concretas en un caso de uso real. ## Competencias validadas - Estructuración de un layout complejo con CSS Grid (áreas nombradas, columnas fluidas, minmax, auto-fit/auto-fill). - Uso de Flexbox para la alineación y distribución interna de componentes (navegación, tarjetas, footer). - Aplicación de consultas de medios (media queries) para adaptar el diseño a múltiples resoluciones. - Combinación coherente de Flexbox y Grid según el contexto de cada sección. - Nombramiento semántico de áreas de cuadrícula y buenas prácticas de organización CSS. - Implementación de una rejilla fluida con repetición automática para la sección de proyectos. ## Estructura mínima esperada de la página La página de inicio del portafolio debe contener las siguientes secciones: 1. **Header / Navegación**: barra de navegación responsiva con logo/nombre y enlaces de menú (usando Flexbox). 2. **Hero / Presentación**: sección de bienvenida con nombre, título profesional y llamada a la...

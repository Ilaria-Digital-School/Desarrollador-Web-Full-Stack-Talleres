# Taller: alineación básica con Flexbox

RECORDATORIO TEÓRICO Flexbox (Flexible Box Layout) es un modelo de diseño CSS que permite distribuir y alinear elementos dentro de un contenedor de forma dinámica y predecible. El modelo mental fundamental es el siguiente: existe un contenedor flex (elemento padre con display: flex) y elementos flex (hijos directos). Sobre el eje principal (por defecto horizontal) actúa justify-content, mientras que sobre el eje transversal (por defecto vertical) actúa align-items. Esta distinción ejes-propiedades es la clave para no confundirse. Propiedades esenciales del contenedor: display: flex activa el contexto flex; flex-direction define la dirección del eje principal (row, column, row-reverse, column-reverse); justify-content alinea los hijos sobre el eje principal (flex-start, flex-end, center, space-between, space-around, space-evenly); align-items alinea los hijos sobre el eje transversal (stretch, flex-start, flex-end, center, baseline); flex-wrap permite que los hijos se distribuyan en varias líneas. Errores frecuentes: confundir justify-content con align-items cuando se cambia flex-direction a column (los ejes se invierten); olvidar que flex-wrap: nowrap es el valor por defecto, lo que puede causar desbordamiento; aplicar propiedades flex a elementos que no son hijos directos del contenedor. Mini-ejemplo: .navbar { display: flex; justify-content: space-between; align-items: center; } Este fragmento distribuye los ítems de una barra de navegación con espacio entre ellos y los centra verticalmente. --- CONTEXTO DEL TALLER Hasta ahora has trabajado con media queries, unidades relativas y los principios del flujo Flexbox, incluida la gestión de márgenes automáticos.

# Taller: cuadrícula básica con 2 columnas y 3 filas

RECORDATORIO TEÓRICO CSS Grid es un sistema de diseño bidimensional que permite controlar simultáneamente filas y columnas, a diferencia de Flexbox, que opera en un solo eje. Un contenedor Grid se activa con display: grid. A partir de ahí, se definen las columnas con grid-template-columns y las filas con grid-template-rows. Por ejemplo, grid-template-columns: 1fr 1fr crea dos columnas de igual ancho; grid-template-rows: 100px 100px 100px define tres filas de 100 píxeles cada una. La unidad fr (fracción) reparte el espacio disponible de forma proporcional. El espacio entre celdas se controla con gap (o column-gap / row-gap por separado). Los elementos hijos directos del contenedor se colocan automáticamente en las celdas, de izquierda a derecha y de arriba abajo, salvo que se especifique lo contrario con grid-column y grid-row. Errores frecuentes: olvidar que grid-template-columns y grid-template-rows se declaran en el contenedor, no en los hijos; confundir el número de líneas con el número de pistas (una cuadrícula de 2 columnas tiene 3 líneas verticales); y usar px fijos cuando fr o % ofrecen mayor flexibilidad. Modelo mental clave: imagina una hoja cuadriculada invisible sobre la página; tú defines cuántas columnas y filas tiene esa hoja, y los elementos se depositan en las celdas resultantes. --- OBJETIVO DEL TALLER Este taller tiene como propósito que construyas, de forma autónoma, una cuadrícula CSS Grid funcional compuesta por 2 columnas y 3 filas, aplicando directamente los conceptos introducidos en la lección teórica anterior sobre terminología de Grid.

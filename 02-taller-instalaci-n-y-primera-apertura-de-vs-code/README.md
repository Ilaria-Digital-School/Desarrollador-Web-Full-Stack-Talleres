# Taller: instalación y primera apertura de VS Code

RECORDATORIO TEÓRICO Un entorno de desarrollo es el conjunto de herramientas que permiten al desarrollador escribir, ejecutar y depurar código de manera eficiente. En el contexto del desarrollo web Full Stack, este entorno incluye tres componentes esenciales que ya hemos identificado: el editor de código (donde se escribe y organiza el código fuente), la terminal (donde se ejecutan comandos del sistema), y el navegador web (donde se visualizan y prueban las aplicaciones). Visual Studio Code (VS Code) es hoy el editor de código más utilizado en el mundo profesional. Se trata de un editor gratuito, ligero y extensible desarrollado por Microsoft, compatible con Windows, macOS y Linux. Su popularidad se debe a su ecosistema de extensiones, su integración nativa con Git y su terminal integrada, lo que lo convierte en una herramienta central del flujo de trabajo del desarrollador. Error frecuente: confundir VS Code (editor de código) con Visual Studio (entorno de desarrollo integrado pesado, distinto y más complejo). Otro error habitual es instalar VS Code sin verificar que la opción 'Agregar al PATH' esté activada durante la instalación, lo cual impide lanzarlo desde la terminal. Modelo mental clave: piensa en VS Code como tu taller de trabajo digital. Antes de construir cualquier cosa, necesitas tener tu taller correctamente montado y conocer la ubicación de cada herramienta.

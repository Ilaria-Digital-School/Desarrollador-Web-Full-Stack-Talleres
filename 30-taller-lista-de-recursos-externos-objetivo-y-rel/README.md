# Taller: lista de recursos externos (objetivo y rel)

**Recordatorio teórico** En HTML, los enlaces externos se crean con la etiqueta y su atributo href. Sin embargo, cuando el destino del enlace es una página o recurso externo, dos atributos adicionales son fundamentales: target y rel. El atributo target controla dónde se abre el enlace. El valor más utilizado es _blank, que abre el recurso en una nueva pestaña o ventana. Existe también _self (por defecto, abre en la misma pestaña), _parent y _top, aunque estos últimos se usan principalmente en contextos con marcos (frames), hoy en desuso. El atributo rel define la relación entre el documento actual y el recurso vinculado. Para enlaces externos, los valores más importantes son noopener (impide que la nueva página acceda al objeto window.opener, evitando un vector de ataque conocido como tabnapping), noreferrer (oculta la URL de origen al sitio de destino e implica noopener) y nofollow (indica a los motores de búsqueda que no sigan ese enlace, útil para contenido no verificado). Error frecuente: usar target="_blank" sin rel="noopener noreferrer". Esto representa un riesgo de seguridad real, ya que la página destino podría manipular la pestaña original mediante JavaScript.

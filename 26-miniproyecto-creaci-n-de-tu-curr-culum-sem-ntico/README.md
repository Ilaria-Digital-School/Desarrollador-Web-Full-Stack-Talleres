# Miniproyecto : creación de tu currículum semántico

## Descripción del proyecto En este miniproyecto, crearás tu propio currículum vitae personal en formato HTML, aplicando todos los principios de semántica, accesibilidad y buenas prácticas aprendidos a lo largo del capítulo 'HTML y estructuración semántica'. Este proyecto es una oportunidad concreta para demostrar que sabes estructurar un documento HTML real, coherente y accesible, tal y como se esperaría de un desarrollador web profesional. El currículum deberá ser completamente funcional como página web autónoma (sin CSS obligatorio, aunque se puede añadir de forma básica). ## Competencias que se validan - Uso correcto de las etiquetas semánticas HTML5 (``, ``, ``, ``, ``, ``, ``, etc.) - Jerarquía de títulos coherente y accesible (H1 → H6) - Respeto de los hitos ARIA básicos para la accesibilidad - Comentarios HTML claros y sangría limpia - Denominación de archivos y estructura de árbol simple - Construcción de una página de columna única bien estructurada ## Contexto profesional Como futuro desarrollador web Full Stack, tu código será leído por otros desarrolladores, evaluado por clientes y analizado por motores de búsqueda y lectores de pantalla. Un HTML semántico, limpio y accesible es la base de cualquier proyecto web profesional. Este currículum será además un primer elemento real de tu portafolio.

# Taller: galería de imágenes responsiva con flex-wrap

**Recordatorio teórico** Flex-wrap es una propiedad CSS de Flexbox que controla si los elementos hijos de un contenedor flexible deben permanecer en una sola línea o pueden distribuirse en varias. Por defecto, flex-wrap tiene el valor nowrap, lo que obliga a todos los elementos a caber en una sola fila, comprimiéndose si es necesario. Con el valor wrap, los elementos que no caben en la línea actual pasan automáticamente a la siguiente, lo que resulta clave para crear diseños responsivos. El valor wrap-reverse hace lo mismo pero en orden inverso. Un modelo mental útil: imagina una caja de huevos. Si tienes 12 huevos y la caja solo tiene 6 espacios por fila, los restantes ocupan una segunda fila. Flex-wrap es exactamente ese mecanismo. La propiedad flex-wrap se combina habitualmente con flex-direction en la propiedad abreviada flex-flow (por ejemplo: flex-flow: row wrap). Para controlar el tamaño mínimo de cada elemento antes de que salte de línea, se usa flex-basis o min-width. Una combinación muy potente para galerías responsivas es: flex-wrap: wrap + flex: 1 1 250px, que permite que cada elemento ocupe al menos 250px y crezca para llenar el espacio disponible. Errores frecuentes: olvidar establecer un flex-basis o min-width adecuado, lo que provoca que los elementos no salten de línea aunque el espacio sea insuficiente; confundir align-content (alinea las filas múltiples) con align-items (alinea los elementos dentro de una sola línea); o no declarar display: flex en el contenedor padre. Mini-ejemplo: .galeria { display: flex; flex-wrap: wrap; gap: 16px; }...

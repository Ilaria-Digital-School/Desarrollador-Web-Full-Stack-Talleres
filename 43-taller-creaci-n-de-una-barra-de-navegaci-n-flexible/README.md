# Taller: creación de una barra de navegación flexible

RECORDATORIO TEÓRICO Flexbox (Flexible Box Layout) es un modelo de disposición CSS diseñado para distribuir elementos dentro de un contenedor a lo largo de un eje principal (horizontal o vertical) y un eje secundario. El contenedor flexible se activa con 'display: flex', lo que convierte a sus hijos directos en elementos flexibles. Las propiedades clave del contenedor son: 'flex-direction' (define el eje principal), 'justify-content' (alinea elementos en el eje principal), 'align-items' (alinea en el eje secundario) y 'flex-wrap' (permite que los elementos se distribuyan en varias líneas). En los elementos hijos, 'flex-grow', 'flex-shrink' y 'flex-basis' controlan cómo cada elemento crece, se reduce o define su tamaño base dentro del espacio disponible. Un error frecuente en principiantes es aplicar propiedades de alineación directamente al elemento hijo en lugar de al contenedor, o confundir 'justify-content' (eje principal) con 'align-items' (eje secundario). Otro error habitual es olvidar que 'flex-wrap: nowrap' es el valor por defecto, lo que puede causar desbordamiento en pantallas pequeñas. Mini-ejemplo: una barra de navegación horizontal básica: nav { display: flex; justify-content: space-between; align-items: center; } Este fragmento distribuye los elementos del menú con espacio entre ellos y los centra verticalmente. Cuando la pantalla se reduce, añadir 'flex-wrap: wrap' junto con una media query permite que los enlaces pasen a apilarse verticalmente, obteniendo un comportamiento responsivo. --- ENUNCIADO DEL TALLER En este taller vas a construir desde cero una barra de navegación completamente flexible y responsiva utilizando exclusivamente HTML semántico y CSS con Flexbox.

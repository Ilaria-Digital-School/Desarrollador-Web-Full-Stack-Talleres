# Taller: primera consulta de medios para cambiar el color de fondo

**Recordatorio teórico** Una media query (consulta de medios) es una regla CSS que permite aplicar estilos de forma condicional según las características del dispositivo o de la ventana gráfica (viewport). Su sintaxis básica es la siguiente: `@media (condición) { /* reglas CSS */ }`. La condición más utilizada en diseño responsivo es `max-width` o `min-width`, que define un punto de interrupción (breakpoint) a partir del cual los estilos se activan o desactivan. El enfoque «mobile first» (prioridad a los dispositivos móviles) consiste en escribir primero los estilos para pantallas pequeñas y luego ampliarlos progresivamente usando `min-width`. Por el contrario, el enfoque «desktop first» usa `max-width` para reducir los estilos a medida que la pantalla se hace más pequeña. Ambos son válidos, pero el enfoque mobile first es hoy la práctica profesional recomendada. Errores frecuentes a evitar: olvidar la etiqueta `` en el HTML (sin ella, las media queries no funcionan correctamente en dispositivos móviles); confundir `max-width` y `min-width` e invertir la lógica de los breakpoints; escribir las media queries fuera del flujo lógico del CSS, lo que genera conflictos de especificidad. Mini-ejemplo: ```css body { background-color: white; /* Estilo por defecto (mobile first) */ } @media (min-width: 768px) { body { background-color: lightblue; /* Pantallas medianas y grandes */ } } ``` Este ejemplo cambia el color de fondo del cuerpo de la página cuando la ventana supera los 768 píxeles de ancho. --- **Enunciado del taller** Este taller es tu primera experiencia práctica con las media queries.

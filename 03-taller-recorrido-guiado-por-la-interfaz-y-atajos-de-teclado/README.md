# Taller: recorrido guiado por la interfaz y atajos de teclado

**Recordatorio teórico** Visual Studio Code (VS Code) es un editor de código fuente desarrollado por Microsoft, gratuito y de código abierto. Se ha convertido en el estándar de la industria para el desarrollo web gracias a su ligereza, su ecosistema de extensiones y su alta personalización. Entender su interfaz no es un detalle cosmético: es una habilidad profesional que impacta directamente tu productividad como desarrollador. La interfaz de VS Code se articula en torno a cinco zonas principales: la Barra de Actividades (izquierda), que da acceso al explorador de archivos, la búsqueda, el control de versiones Git, la depuración y las extensiones; la Barra Lateral, que muestra el contenido del panel activo; el Editor, zona central donde escribes y editas el código; el Panel Inferior, donde se encuentran la Terminal integrada, la consola de salida y los problemas detectados; y la Barra de Estado (inferior), que muestra información contextual como el idioma del archivo, la rama Git activa y los errores. Un error frecuente en principiantes es ignorar la Terminal integrada y seguir usando una terminal externa. La Terminal de VS Code está directamente vinculada al directorio de trabajo del proyecto abierto, lo que evita errores de ruta y agiliza el flujo de trabajo. Otro error habitual es no conocer ningún atajo de teclado y depender exclusivamente del ratón, lo que ralentiza significativamente el trabajo diario.

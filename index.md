Habilidades: diseñar una arquitectura de software segura y multicapa (presentación, negocios, persistencia); desarrollar los componentes de front-end y back-end de una aplicación web; implementar API REST seguras interconectadas a una base de datos relacional o NoSQL; automatizar las implementaciones mediante canalizaciones de CI/CD en una lógica de DevOps; escribir documentación técnica y prepararse para la producción; colaborar en modo de proyectos ágiles (historias de usuario, backlog, sprints, autónomos); -ups);
